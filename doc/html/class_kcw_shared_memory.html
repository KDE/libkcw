Detailed Description

template<typename T>
class KcwSharedMemory< T >

This class is used to abstract a shared memory object.

Author:
Patrick Spendrin
Date:
2011

Shared memory can be used for interprocess communication. For this purpose a KcwSharedMemory object is created on both sides, on one of them create() is called, on the other one you call open(). After that, you can use the memory objects on both sides similar to a pointer to a normal object.
