Detailed Description

C++ Stream based debug output.

Author:
Patrick Spendrin
Date:
2011

KcwDebug can be used to output most values understood by std::ostream (except wide characters!) similar to the following code snippet: 

 KcwDebug() << "the value for x is" << x << endl;
Note:
endl is different from std::endl which cannot be used here.
