Detailed Description

This class is supposed to wrap thread handling.

Author:
Patrick Spendrin
Date:
2011

To use this class, inherit KcwThread and overload run() in your class. If you want to start your thread, just call start() (you don't need to overload that function). The default implementation just starts an eventloop.

Constructor & Destructor Documentation

KcwThread::KcwThread ( HANDLE  exitEventHandle = NULL)

Constructor for a thread object. If you want to override the default event used for signaling the end of the thread, add it as the parameter.

Parameters:
exitEventHandlethe event used for signaling the end of the handle; if exitEventHandle is NULL, a new Event will be created.
Warning:
signaling the exitEvent is currently not threadsave. You should
not change the exitEvent status after you started the thread.
