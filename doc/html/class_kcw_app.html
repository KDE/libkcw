Detailed Description

KcwApp is the Application class which handles the Event distribution and the event loop.

Author:
Patrick Spendrin
Date:
2011

KcwApp is a simple wrapper around KcwEventLoop, which has no additional functionality until now. For future extensions, this class should be used though.
